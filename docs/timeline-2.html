1965

First Wide-area Network(WAN) created

This was established thru a connection via long distant dial-up between a TX-2 computer in MA and Q-32 computer in CA. The system confirmed that packet switching offered most promising model for communication.

1958

Advance Research Project Agency(ARPA) was created
1961

Leonard Kleinrock introduces packet switching

Packet Switching is type of network in which relatively small units of data called packets are routed through a network based on the destination address contained within each packet.
