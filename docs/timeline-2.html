Timeline 2

1965

First Wide-area Network(WAN) created

This was established through a connection via long distant dial-up between a TX-2 computer in MA and Q-32 computer in CA. The system confirmed that packet switching offered most promising model for communication.

1969

IMP network is constructed

Interface Message Processor network is constructed; linked four nodes: University of California at Los Angeles, SRI (in Stanford), University of California at Santa Barbara, and University of Utah; first data packets are sent between networked computers.

1974

Protocol for Packet Network Interconnection published

published by Vint Cerf and Robert Kahn, this specified in detail the design of a Transmission Control Program; the term “Internet” was coined for the first time.

1983

Domain Name System(DNS) is invented

DNS- the phone book of the Internet. We access information online through domain names, like nytimes.com or espn.com. Web browsers interact through Internet Protocol (IP) addresses. DNS translates domain names to IP addresses so the web page can be reloaded.
